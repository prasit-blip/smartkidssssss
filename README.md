# 🎓 ระบบเว็บไซต์โรงเรียนติวเด็กประถม (ครบวงจร)

ระบบจัดการโรงเรียนติวสำหรับเด็กประถม พัฒนาด้วย Python Flask และ Bootstrap 5

## ✨ ฟีเจอร์หลัก

### 🔐 ระบบสมาชิก (Authentication)
- สมัครสมาชิกสำหรับผู้ปกครอง
- เข้าสู่ระบบ/ออกจากระบบ
- รองรับหลายบทบาท: Admin, Parent, Teacher

### 👥 ประเภทผู้ใช้งาน
1. **Admin** - จัดการทุกอย่างในระบบ
2. **Teacher** - เช็คชื่อ, บันทึกผลการเรียน
3. **Parent** - สมัครเรียน, ชำระเงิน, ดูผลการเรียน

### 🏫 ระบบจัดการสาขา (Multi-Branch)
### 👶 ระบบจัดการนักเรียน
### 📚 ระบบคอร์สเรียน
### 🗓️ ระบบตารางเรียน
### 📝 ระบบลงทะเบียนเรียน
### 💳 ระบบชำระเงิน (โอนเงิน + แนบสลิป, Admin อนุมัติ)
### 📍 ระบบเช็คอิน (Attendance)
### 📊 ระบบผลการเรียน

## 🛠️ เทคโนโลยีที่ใช้
- Backend: Python Flask
- Database: SQLite (SQLAlchemy)
- Frontend: HTML5, Bootstrap 5, Thai Font (Sarabun)

## 📦 การติดตั้ง
```bash
pip install -r requirements.txt
python app.py
```
